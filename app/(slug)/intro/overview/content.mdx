# What are the prerequisites?
You don't need to know anything about SQL to start with this workshop. But you should have a basic understanding of programming.

You will need Postgres installed on your machine. You can download it from [here](https://www.postgresql.org/download/)

&nbsp;

# What will you learn?

You will learn how to use SQL to store, query, and manipulate data. We will start with the basics and gradually move to more advanced topics. At the end of the workshop, you will be able to build a simple database application.

So let's get started.
